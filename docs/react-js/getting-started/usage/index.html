Uso

Os componentes Stardust foram construídos abstraindo o máximo de configurações, otimizando o tempo de customização!

Mas também foram pensados para deixar esta customização fácil e intuitiva. Todos os componentes e funções podem ser importados de @stardust-ds/react, pois utilizamos Exportações Explicitas.

Vamos começar​

O uso dos componentes é realmente simples:

  • Importar o componente:
  import { Button } from '@stardust/react'
  • Após feita a importação, é apenas usá-lo:
Editor em tempo real
function MyPlayground(props) {
  return (
    <div>
      <Button label='Botão exemplo' />
    </div>
  );
}
